Machine Learning! Where to start?

Author | Sumit Agarwal Category | Technology
2018 Jun 04

Wikipedia defines Machine Learning as :

Machine learning is a field of computer science that often uses statistical techniques to give computers the ability to "learn" (i.e., progressively improve performance on a specific task) with data, without being explicitly programmed.

Tom M. Mitchell provided a widely quoted, more formal definition of the algorithms studied in the machine learning field:

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E.

Machine Learning is one of the most recent topics that has kept the humans guessing as to what extent can machines behave as humans. It has kept everyone interested in knowing more about it, to know what are the different algorithms and approaches and what is the exact logic behind them. Here I try to list down all the possible and best resources to kick start machine learning and even master topics such as applications of machine learning in vision (using convolutional neural networks), applications in natural language processing (using recurrent neural networks).

You can start with the very famous Andrew Ng's Machine Learning course on Coursera. It's an introductory course which very slowly and steadily covers the basic topics like Supervised learning, Linear Regressions, Logistic Regressions, Support Vectore Machines, Neural Networks, Large Scale Machine Learning, etc.
